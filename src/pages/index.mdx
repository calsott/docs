# Welcome to Calsott!

## What is Calsott?

Calsott is a complete open-source tool that allows you to monitor the performance of your application in a Continuous Integration environment.

## Why Calsott?

It uses official [Lighthouse](https://github.com/GoogleChrome/lighthouse) tool for running the audits, so you can be sure that the results are reliable.

It **adapts to "any" CI environment** , and it is easy to integrate with your existing tools. See the [available CI adapters compatibility](/adapters)

It send the results to a reliable providers like Datadog, New Relic, or Google Analytics. That tools 
enables you to monitor your application performance in a reliable way, also you could create alerts and dashboards on them.

These three points offer you **reliability, flexibility, scalability and security**.
